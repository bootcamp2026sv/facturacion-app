Estructura de Formularios: Bootstrap vs PrimeFlex

Descubre cómo agrupar, alinear y distribuir campos en un formulario usando las utilidades de PrimeFlex.

1. Agrupación Vertical Básica (Label + Input)

Concepto: .field vs .mb-3
• En Bootstrap 5, los inputs se agrupan verticalmente usando la utilidad de margen inferior .mb-3.
• En PrimeFlex, existe una clase semántica llamada .field que añade automáticamente el margen inferior idóneo y el espaciado necesario para los campos.
Bootstrap (.mb-3 + .form-label + .form-control)
<div class="mb-3">
  <label class="form-label">Usuario</label>
  <input type="text" class="form-control">
  <div class="form-text">No compartas tu usuario.</div>
</div>
No compartas tu usuario.
PrimeFlex (.field + .block + inputs)
<div class="field">
  <label class="block mb-2 font-medium">Usuario</label>
  <input type="text" class="w-full p-2 border-round ...">
  <small class="block mt-1 text-gray-500">No compartas tu usuario.</small>
</div>
No compartas tu usuario.

2. Layout en Columnas (Formulario en Rejilla)

Concepto: .formgrid.grid vs .row
• En Bootstrap, para poner dos campos en la misma línea usas .row con columnas individuales y espaciado (ej. .col-md-6).
• En PrimeFlex, es mejor utilizar la clase .formgrid en combinación con .grid. Esto optimiza los márgenes y paddings internos especialmente para inputs, y dentro aplicas .field.col-12.md:col-6.
Bootstrap (.row > .col-md-6 > .mb-3)
<div class="row">
  <div class="col-md-6 mb-3">
    <label class="form-label">Nombre</label>
    <input type="text" class="form-control">
  </div>
  <div class="col-md-6 mb-3">
    <label class="form-label">Apellido</label>
    <input type="text" class="form-control">
  </div>
</div>
PrimeFlex (.formgrid.grid > .field.col-12.md:col-6)
<div class="formgrid grid">
  <div class="field col-12 md:col-6">
    <label class="block mb-2">Nombre</label>
    <input type="text" class="w-full p-2 ...">
  </div>
  <div class="field col-12 md:col-6">
    <label class="block mb-2">Apellido</label>
    <input type="text" class="w-full p-2 ...">
  </div>
</div>

3. Layout Horizontal (Etiquetas alineadas a la izquierda)

Concepto: .field.grid vs .row.align-items-center
Para posicionar la etiqueta al lado del campo de entrada:
• En Bootstrap, se usa .row en el contenedor y se añaden clases de columnas tanto al <label> como al contenedor del input.
• En PrimeFlex, se agrega la clase .grid directamente al contenedor .field (quedando .field.grid) y luego se define la anchura del label (ej: col-12 md:col-2) y la del input (ej: col-12 md:col-10).
Bootstrap (.row + .col-form-label)
<div class="row mb-3 align-items-center">
  <label class="col-sm-3 col-form-label">Email</label>
  <div class="col-sm-9">
    <input type="email" class="form-control">
  </div>
</div>
PrimeFlex (.field.grid > label + .col)
<div class="field grid align-items-center">
  <label class="col-12 md:col-3 font-medium">Email</label>
  <div class="col-12 md:col-9">
    <input type="email" class="w-full p-2 ...">
  </div>
</div>

4. Checkbox y Alineación de Opciones

Concepto: .field-checkbox vs .form-check
• En Bootstrap, los checkboxes y radios usan la estructura .form-check para alinear correctamente el input y el label.
• En PrimeFlex, existe una clase semántica específica .field-checkbox (o .field-radiobutton) que se encarga de aplicar los márgenes, flex y alineación vertical óptima de forma nativa.
Bootstrap (.form-check)
<div class="form-check mb-3">
  <input type="checkbox" class="form-check-input" id="chk1">
  <label class="form-check-label" for="chk1">Recordarme</label>
</div>
PrimeFlex (.field-checkbox)
<div class="field-checkbox">
  <input type="checkbox" id="chk2">
  <label for="chk2">Recordarme</label>
</div>

Resumen de Clases de Formulario

Propósito del Diseño Bootstrap 5 PrimeFlex v4 Recomendación de Uso
Contenedor de Campo Vertical .mb-3 .field Añade espaciado de margen inferior idóneo para agrupar etiqueta y campo.
Grid del Formulario .row .formgrid.grid PrimeFlex usa `.formgrid.grid` para que las columnas de los campos se espacien mejor.
Contenedor Horizontal .row + cols .field.grid Aplica rejilla directamente sobre el campo para alineación en la misma fila.
Checkbox .form-check .field-checkbox Alinea automáticamente el checkbox con el texto de la etiqueta en flexbox.
Radio Button .form-check .field-radiobutton Igual que el checkbox pero especializado en botones de opción redondos.
Texto de ayuda/secundario .form-text .block.mt-1.text-xs (o similar) PrimeFlex confía en utilidades genéricas de tipografía y espaciado.
